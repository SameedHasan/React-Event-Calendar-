# react-event-calendar-suite

Interactive demos for the calendar component. Use the **Calendar** stories to explore views, controlled mode, theming, and customization slots.

## Install

```bash
npm install react-event-calendar-suite
```

Peer dependencies (install in your app):

```bash
npm install react react-dom antd @ant-design/icons dayjs zustand
```

## Quick start

```jsx
import Calendar from 'react-event-calendar-suite';
import 'react-event-calendar-suite/style.css';

const events = [
  {
    id: 1,
    title: 'Team sync',
    start: new Date(2026, 5, 9, 9, 0),
    end: new Date(2026, 5, 9, 10, 0),
    type: 'Meeting',
  },
];

export function App() {
  return <Calendar events={events} defaultView="month" />;
}
```
